Grammar Block
-에 / -에서
Location particle. 에 for static location (at, in), 에서 for dynamic location (from, at for action).
집에 있어요 — I am at home.
Expression Block
-이건 / -그건
Contraction of 이것은 / 그것은. Common in spoken Korean.
이건 뭐예요? — What is this?
